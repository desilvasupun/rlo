Ubuntu Server + Production-Ready LAMPP Stack RLO
===============================================

Open index.html in a web browser to use the learning object.

Structure
---------
- index.html: overview, learning outcomes, module map
- modules/: step-by-step hands-on modules
- references.html: implementation notes and sources
- styles.css / script.js: offline styling and interaction
- assets/architecture.svg: stack diagram

Notes
-----
- Checklist progress is stored locally in the browser only.
- The site is fully static and can be zipped and submitted as the actual RLO.
