Implementation notes and sources

This RLO was built around the supplied design document and aligned to current official vendor documentation consulted on March 11, 2026. The module sequence, verification checkpoints, and learner workflow come from the design document. The commands and configuration patterns were cross-checked against current Ubuntu, MariaDB, and Python documentation.

Reuse guidance

  • When you reuse this RLO later, verify package names and screenshots against the current Ubuntu Server LTS release at that time.
  • Keep the modular sequence intact so learners can validate one layer before moving to the next.
  • If you later add HTTPS, a reverse proxy, or deployment automation, treat those as separate advanced modules rather than expanding the baseline build too far.
